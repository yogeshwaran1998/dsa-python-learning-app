# Searching Algorithms - Theory Guide

## Binary Search

Binary search finds element in O(log n) time on sorted arrays.

### Time Complexity
- Time: O(log n)
- Space: O(1) iterative, O(log n) recursive

### Key Variations
1. **Lower bound**: First element >= target
2. **Upper bound**: First element > target
3. **Search in rotated array**
4. **Search in unknown search space**

## When to Use Binary Search

- Sorted data
- Finding boundary conditions
- Monotonic functions
- Optimization problems with monotonic objective
